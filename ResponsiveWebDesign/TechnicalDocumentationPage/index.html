Object Oriented Programming

Java is known as a Object Oriented Programming Language

It means it is easy to represent objects by classes and encapsulation

Here is an example: 

                    
                        
                        // Car.java file
                        // public class
                        public class Car {
                            // public variable
                            public int tireCount;

                            // public method
                            public void display() {
                                System.out.println("I am a Car.");
                                System.out.println("I have " + tireCount + " tires.");
                            }
                        }

                        // Main.java
                        public class Main {
                            public static void main( String[] args ) {
                                // accessing the public class
                                Car car = new Car();

                                // accessing the public variable
                                car.tireCount = 4;
                                // accessing the public method
                                car.display();
                            }
                        }

The output of this code will be:

I am a Car. I have 4 tires.
